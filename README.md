## Usage
「脚」などのセクションを作成し、

セクション内に「レッグプレス」などの種目カードを作成します。

種目カード内では、インターバルを設定したり、

重さ×回数を入力したりすることができます。

また、コメントを入力したり、お気に入りマークをつけることもできます。
